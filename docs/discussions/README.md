# OpenO1 讨论记录

这个目录用于保存 OpenO1 项目推进过程中形成的重要讨论记录。

记录范围包括但不限于：

- 推理框架设计
- AgentTeam 协作方式
- 主 agent / subagent 职责边界
- 可验证委托协议
- 共享上下文与长期记忆
- 数学推理验证机制
- 代码、搜索、文档等任务类型的扩展方案
- 项目阶段性架构决策

## 记录原则

1. 每次重要讨论单独保存为一个 Markdown 文件。
2. 文件名使用 `YYYY-MM-DD-topic.md` 格式。
3. 记录重点放在问题、结论、设计取舍和后续行动上。
4. 不追求逐字转录，优先沉淀可执行的项目知识。
5. 对尚未验证的判断，要标记为假设或待验证事项。

## 推荐模板

```md
# 标题

- 日期：YYYY-MM-DD
- 类型：架构讨论 / 实现方案 / 研究记录 / 决策记录
- 状态：草案 / 已采纳 / 待验证 / 已废弃

## 背景

## 核心问题

## 讨论结论

## 设计方案

## 风险与争议

## 后续行动
```
